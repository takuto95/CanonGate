# Spec: ステータス（#ステータス）

## 背景 / 目的
ユーザーの現在状態（設定/タスク/ゴール/統計/推奨アクション）を、LINE上で番号選択式に確認できるようにする。

## UI（LIFF）
- リッチメニューの「📊 ステータス」は **LIFF表示**を優先する（`https://liff.line.me/2008835190-F0ZpGhEt/status`）。
- 既存の `#ステータス` コマンド（番号選択式）は継続する（チャット内で確認したい場合のフォールバック）。

## コマンド
- `#ステータス` / `#状態` / `#status` / `ステータス`

## 動作（要点）
- メニューを表示し、ユーザーは番号（1〜8、0=終了）で選ぶ
- どのモード中でも **設定やstatus/split/retry等の優先コマンドは割り込みで実行**される（SSoT参照）

## メニュー項目（要約）
1. パーソナライズ設定
2. 今日のタスク
3. ゴール進捗
4. タスクサマリー
5. 最近の活動（ストリーク）
6. 統計情報
7. 推奨アクション
8. すべて表示
0. 終了

## API
- `GET /api/status?userId=<userId>&format=<text|json>`

## 受入条件（Given/When/Then）
- Given: ユーザーが `#ステータス` を送る
- When: 1〜8 の番号を送る
- Then: 選択したセクションの内容が返る
